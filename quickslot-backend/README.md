# Quick-Slot Backend

This is the backend service for the Quick-Slot parking application.

## Structure

- `/config` - Database and external services configuration
- `/middleware` - Custom Express middlewares (like authentication)
- `/routes` - API route definitions
- `server.js` - Main entry point

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start development server:
   ```bash
   npm run dev
   ```
